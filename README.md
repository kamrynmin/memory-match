# Memory Match



## Technologies Used

- HTML5
- CSS3
- jQuery
- AWS EC2

## Live Demo

Try the application live at [https://memorymatch.kamrynmin.com](https://memorymatch.kamrynmin.com)

## Features

- User can win by matching all the card pairs.
- User can shuffle cards.
- User can reset the game.
